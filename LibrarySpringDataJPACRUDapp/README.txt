Модифицированное приложение для цифрового учета книг. 
Во многом по функционалу повторяет первый проект, но JDBCTemplate заменен на Hibernate и Spring Data JPA.

Новый функционал:
1) Добавлена возможность пагинации книг.
2) Добавлена сортировка книг по году.
3) Добавлена страница поиска книг.
4) Добавлена автоматическая проверку на то, что человек просрочил возврат
книги.

Используемые технологии:
Spring MVC, Thymeleaf, Hibernate, Spring Data JPA
